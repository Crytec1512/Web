Добрый день!


Проект следует запускать из папки web-project, чтобы все пути корректно подхватились
Проект запускается с помощью скрипта ./setup.sh

В данном проекте присутствуют следующие файлы:

api/API.py - микросервис для взаимодействия с БД
frontend/build - сбилженный статический сайт REACT
bd/BD.sql - база данных и её заполнение
nginx/server.conf - конфиг для nginx сервера
package.json - полный список зависимостей для JS
requirements.txt - список зависимостей из venv для python
setup.sh - запуск проекта
PG_SUPERUSER_PASSWORD=<password for postgres> ./delete.sh - Команда для очистки созданных конструкций
в .gitignore находятся модули для JS и venv
--------------------------------------------------------
В setup.sh подключается venv устанавливаются зависимости для python автоматически,
если такая опция не нужна, лучше закомментировать.
